# コントリビューションガイド

Issue Triage Action へのコントリビューションを歓迎します！🎉

## 開発環境のセットアップ

```bash
git clone https://github.com/hihutuka/issue-triage-action.git
cd issue-triage-action
npm install
```

## 開発フロー

1. `main` ブランチから新しいブランチを作成してください
2. 変更を加え、テストが通ることを確認してください
3. Pull Request を作成してください

## コーディングガイドライン

- JavaScript (ES2020+) で記述してください
- 関数には JSDoc コメントを付けてください
- テストを追加してください

## Issue の報告

- バグ報告には再現手順を含めてください
- 機能リクエストにはユースケースを記述してください
